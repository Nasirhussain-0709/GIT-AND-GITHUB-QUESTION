Q.2:- What do you understand by the term 'version control system'?

Ans :- Version control also known as source control, is the practice of tracking and managing changes to software code. 	Version control system are software tools that help software team manage changes to source code over time.

Q.3:- What is GitHub?

Ans :- GitHub allows you to create, store, change, merge, and collaboration on files or code. Any member of a team can access 	the GitHub repository and see the most recent version in real time. Then, they can make edits or changes that the 	other collaborators also see.
